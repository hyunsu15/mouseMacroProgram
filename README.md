# exe파일추가

2019.7.15 exe파일을 추가함.  
파이썬이 없는 사람도 할수있도록 실행파일을 추가함.  
autoGui의 딜레이는 0.045임.  
코드를 수정하여 입맛대로 해도 됨.

# 설명

- LD플레이어로 집형검키우기를 킨다.
- 왼쪽화면에 맞게 화면을 맞춘다.
- exe파일 을 실행하여 start 버튼을 누른다.
